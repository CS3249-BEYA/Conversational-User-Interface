1. Install Python (if needed)
brew install python@3.11

2. Install Ollama
brew install ollama

3. Start Ollama service
ollama serve

4. Pull the model (in another terminal tab)
ollama pull phi3:medium

5. Test run
ollama pull phi3:medium "Hello"

6. set up environment
python3 -m venv venv
source venv/bin/activate

7. install requirements
pyenv local 3.12.2
pip --version
pip install -r requirements.txt

8. starting the app
python -m app.app

9. go to http://127.0.0.1:5000/ on ur web browser to talk to the Chatbot
